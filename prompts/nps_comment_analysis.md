# NPS Comment Analysis – Standard AI Prompt

## Objective
Analyze customer comments from NPS surveys to extract actionable insights.

## Input Data
You will receive:
- Customer comments (verbatim)
- NPS score (0–10)
- Client name (if available)
- Domain (Data & Analytics, Population Health, Provider, Pharmacy)
- State (US postal code)

## Tasks
1. Classify each comment as:
   - Positive
   - Neutral
   - Negative

2. Identify key themes such as:
   - Product functionality
   - Service quality
   - Timeliness
   - Communication
   - Support
   - Pricing (if mentioned)

3. Tag sentiment drivers (what caused the sentiment).

4. Highlight:
   - Quick wins
   - Systemic issues
   - Client-specific risks

## Output Format
Provide results in a table with columns:
- Client
- State
- Domain
- NPS Score
- Sentiment
- Key Theme
- Insight Summary
- Recommended Action

## Tone
- Clear
- Business-focused
- Action-oriented
